hex triplet
A hex triplet is a six-digit, three-byte hexadecimal number used in HTML, CSS, SVG, and other computing applications to represent colors.
The bytes represent the red, green and blue components of the color.
One byte represents a number in the range 00 to FF (in hexadecimal notation), or 0 to 255 in decimal notation.
This represents the least (0) to the most (255) intensity of each of the color components. Thus web colors specify colors in the 24-bit RGB color scheme.
Thus, there are 24 bits used to specify a web color within the sRGB gamut, and 16,777,216 (224) colors
The RGB color model is an additive color model in which red, green, and blue light are added together to reproduce a broad array of colors
Complementary colors are opposite each other on the color wheel
triads are a set of three colors forming an equilateral triangle on the color wheel.

Notes
secondary color
A secondary color is an even combination of any two primaries. Yellow is #FFFF00
tertiary color color
A tertiary color or intermediate color is a color made by mixing full saturation of one primary color with half saturation of another primary color and none of a third primary color, in a given color space such as RGB (wikipedia)
Chartreuse is 7FFF00, "orangieis FF7F00
Subtractive CMY
The CMY color model is a subtractive color model in which cyan, magenta and yellow pigments or dyes are added together in various ways to reproduce a broad array of colors
transparent
The transparent keyword represents a fully transparent color. This makes the background behind the colored item completely visible.
Technically, transparent is a shortcut for rgba(0,0,0,0)
rgb()
white may be represented by rgb(255,255,255)or rgb(100%,100%,100%)
mnemonic
#000000 and rgb(0,0,0) are black: A black hole
currentcolor
The currentcolor keyword represents the value of an element's color property.
This lets you use the color value on properties that do not receive it by default.
