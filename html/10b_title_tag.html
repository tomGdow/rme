<title>
Defines the document title which is shown in browser title bar or page tab
Syntax
  1. <title>My Wonderful Site</title>

Examples
Notes
Type of element
title is an escapable raw text element, which means it can have text and character references, but the text must not contain an ambiguous ampersand (or contain other tags)
Validation
The title element (which must be in the head section) is essential for HTML validation
There may only be one title element per document (in head)
Is a significant factor in SEO
50-60 characters max, but make unique
May only contain text - no tags
It may also be used used in bookmarks, title bar, nav bar, history
