details/summary (disclosure)
xx
details/summary
The HTML Details Element creates a disclosure widget in which information is visible only when the widget is toggled into an "open" state.
A summary or label can be provided using the summary element.
disclosure widget is typically presented onscreen using a small triangle which rotates (or twists) to indicate open/closed status, with a label next to the triangle.
If the first child of the details element is summary, the contents of the summary element are used as the label for the disclosure widget.
open
This Boolean attribute indicates whether or not the details - that is, the contents of the details element - are currently visible.
The default, false, means the details are not visible.
