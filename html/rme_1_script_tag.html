Script Tag
xx
  • The HTML <script> tag is used to embed or reference code
Syntax
  1. <script src="javascript.js"></script>

Examples
  1. <script type="text/javascript" src="javascript.js"></script>
  2. <script src="javascript.js"></script>
  3. <script>
      alert("Hello World!");
    </script>
Notes
  1. defer

    This Boolean attribute indicates to browser that the script is meant to be executed after the document has been parsed, but before firing DOMContentLoaded.
