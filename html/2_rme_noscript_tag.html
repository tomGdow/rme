noscript
The HTML <noscript> element defines a section of HTML to be inserted if a script type is unsupported or if scripting is currently turned off
Syntax
  1. <noscript></noscript>

Examples
  1. <noscript> <a href="https/www.mozilla.com">Ext Link</a> </noscript>
Notes
  1. Both starting and ending tags are mandatory
