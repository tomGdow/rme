xx
head
Contains machine-readable information ( metadata ) about the document such as title, scripts, style sheets
HTML5-compliant browsers automatically create a <head> element if its tags are omitted in the markup.
Zero or more elements of metadata content, of which no more than one is a title element and no more than one is a base element. ( W3C )
Otherwise: One or more elements of metadata content, of which exactly one is a title element and no more than one is a base element. ( W3C )
Syntax
  1. <html></html>
