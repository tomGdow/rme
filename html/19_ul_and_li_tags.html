ul
Represents an unordered list of items
Syntax
unordered list
<ul>
<li></li>
<li></li>
<li></li>
</ul>
Examples
html-1
<ul class="square-bullets">
<li>Javascript</li>
<li>Cascading Style Sheets</li>
<li>C Programming</li>
</ul>
html-2
<ul class="image-bullets">
<li>Javascript</li>
<li>Cascading Style Sheets></li>
<li>C Programming</li>
</ul>
css
ul.square-bullets {
list-style-type: square;
float:left;
}
ul.image-bullets {
list-style-image: url(http://xu/tgdfiles/png/go.png);
clear:both;
}
Notes
list-style-type
disc
circle
square
none
list-style-image
list-style-image: url(my-url); // CSS
Markers
Markers are in left padded area of content box
list-style
list-style is a shorthand for list-style-type, list-style-image and list-style-position
