ul
Represents an unordered list of items
Syntax
html
<ul>
<li></li>
<li></li>
<li></li>
</ul>
alternative syntax
<ul>a
<li>b
<li>c
<li>
</ul>
Examples
html-1
<ul class="square-bullets">
<li>Javascript</li>
<li>Cascading Style Sheets</li>
<li>C Programming</li>
</ul>
html-2
<ul class="image-bullets">
<li>Javascript
<li>Cascading Style Sheets
<li>C Programming
</ul>
css
ul.square-bullets {
list-style-type: square;
float:left;
}
ul.image-bullets {
list-style-image: url(http://xu/tgdfiles/png/go.png);
clear:both;
}
Notes
list-style-type
disc
circle
square
none
list-style-image
list-style-image: url(my-url); // CSS
Markers
Markers are in left padded area of content box
list-style
list-style is a shorthand for list-style-type, list-style-image and list-style-position
closing tag
the closing tag </li> is optional.
An li element's end tag may be omitted if the li element is immediately followed by another li element or if there is no more content in the parent element.
