Notes
Mime type
The mime type is text/vtt
VTT file comments
Note signifies a comment
Comments may contain newlines but not blank lines (which signify close comment)
&, -->, >, must be escaped in comments
Tags in VTT text
c (class), i (italic), b (bold), v (voice) may all be included in cue text
Time format
Note that time should be in the format hh:mm:s.xxx or mm:s.xxx and that is dot-milliseconds (not semi-colon)
