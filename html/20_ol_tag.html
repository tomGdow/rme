ol
An ordered list, typically rendered as a numbered list
Syntax
html
<ol start="1">
  <li></li>
  <li></li>
  <li></li>
</ol>
Alternative Syn
<ol start="1">
  <li>a
  <li>b
  <li>c
</ol>
Examples
html
<ol id="list-2" start="4">
  <li>Lion</li>
  <li>Springbok</li>
  <li>Brown Haret;/li>
css
ol#list-2{
  list-style-type:upper-roman;
}
Example 2
<ol type="I" start="4" reversed>
  <li>First Item</li>
  <li>Second Item</li>
  <li>Third Item</li>
</ol>
Notes
Six types of Automatic numbering
Decimal, Roman, Latin, Greek, Georgian, Armenian,
markup
decimal
decimal-leading-zero
lower-roman
upper-roman
lower-roman
lower-latin (lower-alpha)
upper-latin (upper-alpha)
lower-greek
armenian
georgian
latin vs alpha
upper-alpha and upper-latin are identical, as are lower-latin and lower-alpha
closing tag
the closing tag </li> is optional.
An li element's end tag may be omitted if the li element is immediately followed by another li element or if there is no more content in the parent element.
start
Value is always an Arabic number
type
a is lower-case letter
A is upper-case letter
i is Roman numeral (lower)
I is Upper-Roman
1 is decimal (default)
reversed
<ol reversed >
reverses the order
value vs start
It seems that value may be used instead of start as an attribute of the first li tag
<ul value="2">
(MeasureUp, but idiosyncratic)
