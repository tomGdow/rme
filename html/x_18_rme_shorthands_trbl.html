Notes
Caveat One
Values not specified are set to initial value
Box Edges
1-Value syntax
values apply to all four edges
2-Value syntax
mnemonic: 2+
1-[top,bottom] 2-[left,right]
3-Value syntax
mnemonic 3-
1-top 2-[left-right] 3-bottom
4-Value syntax
mnemonic TRBL
mnemonic clockwise
1-top 2-right 3-bottom 4-left
Box Corners
1-Value syntax
All four corners have the same value
2-Value Syntax
mnemonic 2X
1-[TopLeft,btmRight] 2-[topRight,BtmLeft]
3-Value Syntax
mnemonic 3Z
1-[TopLeft] 2=[topRight,BtmLeft] 3-[btmRight]
4-Value Syntax
mnemonic Clockwise from top left
