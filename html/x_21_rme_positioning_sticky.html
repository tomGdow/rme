Sticky
A stickily positioned element is treated as relatively positioned until its containing block crosses a specific threshold within its flow root (or the container it scrolls within), at which point it is treated as 'stuck' until meeting the opposite edge of its containing block (ie until it reaches the boundary of its parent).
Syntax
css
.stky {
position: sticky;
top: 0;
}

Notes
Containing block
For position sticky, the containing block is the same as that for position relative
Positioned element
A positioned element is an element whose computed position is one of relative, absolute, fixed or sticky.
In other words, it's anything except static
