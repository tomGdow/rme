It lets you specify timed text tracks (or time-based data), for example to automatically handle subtitles `

Notes
default
indicates that the track should be enabled unless the user's preferences indicate that another track is more appropriate
Must be enabled on at least on track, it seems
(But only on one track)
kind
How the text track is meant to be used. If omitted the default kind is subtitles
Values for kind attr
Value Description Comment
subtitles Provides a translation of context that cannot be understood by the viewer default
captions transcription and possible translation May include audio cues
descriptions Textual description of content Suitable for visually impaired
chapters Titles suitable for when user is navigating media
metadata tracks used by scripts. Not visible to user
Note that all except metadata are plural
label
A user-readable title of the text track which is used by the browser when listing available text tracks.
src
Address of the track (.vtt file)
Must be a valid URL
must be specified
srclang
Language of the track text data
It must be a valid BCP 47 language tag
If the kind attribute is set to subtitles, then srclang must be defined.
