Syntax
Method 1
display:table-cell
#my-div {
    display:table-cell; 
    vertical-align:middle;
    text-align-center;
    height: 10vh;
    width: 10vh;
}
Method 2
line-height: height-of-div
#my-div {    
    line-height: 10vh;
    text-align: center;
    display: inline-block;
    width: 10vh;
}
Method 3
Flexbox 
#my-div {             
    display:flex;
    width: 10vh;
    height: 10vh;
    justify-content: center;
    align-items: center;
}
Method 4
Align div vertically and horizontally

    margin: 0 auto;
}

    position:absolute;
    top:50%;
