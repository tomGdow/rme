Table . CSS Units of Measurement
Unit Description Comment
Absolute lengths
cm centimetres
mm millimeters
in inches 1in = 96px = 72pt = 2.54cm
px pixels 1 px = 1/96 of 1in
pt point 1 pt = 1/72 in
pc picas 1 pc = 12 pt
Q Quarter-millimeters 1Q = 1/40th of 1cm
Relative lengths
em Relative to the font-size of the parent element in terms of typographical elements like font-size
em (unlike percentage) is always relative to font-size.
Relative to the font-size of the element itself in terms of other properties such as width
ex relative to the x-height of element's font rare
ch relative to the width of of the glyph "0" (zero) of the element's font.
rem relative to the root element
vw Relative to 1% of the width of viewport
vh Relative to 1% of the height of viewport
vmin Relative to 1% of viewport's smaller dimension
vmax relative to 1% of viewport's larger dimension
% relative to the parent element
if you set an element's font-size as a percentage it will be a percentage of the font-size of the element's parent.
Percents depends on context. For example, if they're used in font-size, they will be relative to the current font-size;
if it's in height, they will be relative to the height.
If you use a percentage for a width value, it will be a percentage of the width of the parent.
lh line-height relative to the line-height of the element
