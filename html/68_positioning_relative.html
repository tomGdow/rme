Relative
The element is positioned according to the normal flow of the document
The element is then offset relative to itself based on values of top, right, bottom, left
The offset does not affect other elements
Thus the space given to the element in the page is the same same as if the position was static
Syntax
css

.relative {
    position: relative;
    top: 3vw;
    left: 5vw;
}
Examples
Opening Example

<div class="rel-container">
    <div id="div-one">Static</div>
    <div id="div-two" class="relative">Relative</div>
    <div id="div-three">Static</div>
</div>

js 
function toggle(id) {
    if (document.getElementById(id)) {
        document.getElementById(id).addEventListener('click', function () {
            if (this.classList.contains('relative')) {
                this.classList.remove('relative');
                this.innerHTML = "Static";
            } else {
                this.classList.add('relative');
                this.innerHTML = "Relative";
            }
        });
    }
}
