Meta tag
xx
Meta data
Represents meta data that cannot be represented by other HTML meta elements such as base, link, script, style, title
Syntax
  1. <meta name="" content="">
  2. <meta http-equiv="" content="">
  3. <meta charset="">
Examples
  1. <meta charset="utf-8">
  2. <meta http-equiv="X-UA-Compatible" content="ie=edge">
  3. <meta http-equiv="refresh" content="5">
  4. <meta http-equiv="refresh" content= "5; url='my_new_page.html' ">
  5. <meta name="viewport" content="width=device-width,initial-scale=1.0">
  6. <meta name="keywords" content="my,keywords">
  7. <meta name="description" content="my description">
  8. <meta name="robots" content="index, follow">
  9. <meta name="robots" content="nofollow">
Notes
Content
Value for http-equiv or name attribute
name
Possible values are author, description, keywords, referrer, (and then content="no-referrer", for example), and creator
http-equiv
Defines a Pragma directive
Called http-equiv because all allowed values are names of http headers.
Pragma directive
Information given by Server about how Web Page is served.
refresh
Positive integer. The number of seconds until page reloads/redirects
May be followed by ';url="myurl'
uft-8
Uniform Transformation Code
utf-8 alternatives
<meta charset="utf-8">
<meta charset="big5">
<meta charset="US-ASCII">
