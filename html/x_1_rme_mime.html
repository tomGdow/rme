Mime Type
Multipurpose Internet Mail Extension
Indicates the nature and format of a document
Syntax
  1. type/subtype
  2. type/subtype;parameter=value;

type is general category

subtype is exact type of data

Examples
  1. text/html
  2. text/plain;charset=UTF-8
  3. <form action="http://localhost:8000/" method="post" enctype="multipart/form-data">
Notes
  1. Two classes of type: discrete and multipart
  2. Generic binary data has type of application/octet-stream
  3. Multipart: Composite document broken into pieces, each of which may have individual MIME type
