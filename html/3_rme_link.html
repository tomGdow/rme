link
The HTML External Resource Link Element or <link> specifies the relationship between the current document and an external resource
Usually, links to stylesheet but may also link to icons, such as favicon
Can also include media query
May occur in body or head (stylesheet is 'body-OK') but not good practice.
Syntax
  1. <link href="main.css" rel="stylesheet">
  2. <link href="bonnie.ico" rel="icon">
Examples
  1. <link href="print.css" rel="stylesheet" media="print">
  2. <link href="mobile.css" rel="stylesheet" media="screen and (max-width: 600px)">
  3. <link rel="apple-touch-icon-precomposed" sizes="114x114" href="apple-icon-114.png" type="image/png">
  4. <link rel="icon" href="http://xu/tgdfiles/icon/bonnie.ico">
Notes
sizes
Specifies icon size (sizes="114x114)
type
Specifies MIME type (type="image/png")
title
Special semantic meaning in link tag. Specifies a preferred or alternative stylesheet
disabled
For rel="stylesheet" only
preload
If rel="preload" then may use as attribute
rel
The rel attribute (and the href attribute ) is compulsory.
Other values for rel are author, icon, stylesheet, alternate.
hrefetymology
hrefstands for hypertext-reference
Tim Benners-Lee: "A" is for "anchor", "HREF" is for "hypertext reference"
