border (shorthand)
The border shorthand CSS property sets an element's border.
It sets the values of border-width, border-style, and border-color
border-stylemust be specified as it defaults to none
Thus border:solid is enough to render a border
However border: red thickis insufficient
dotted dashed solid double groove ridge are all examples of border-style
