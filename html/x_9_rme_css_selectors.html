CSS selectors
Define the elements to which a set of CSS rules apply
Syntax
  1. code

Examples
  1. a[title] {color: purple; }
  2. a[href="https://example.org"]
  3. a[href*="example"]
    // href attr containing example anywhere in the string
  4. a[href$=".org"]
    5. // href attr ending in org
  5. a[class~="logo"]
    6. class attr that contains the word logo anywhere in the space-separated string
  6. a[href*="insensitive" i]
    7. case-insensitive comparison
  7. A * B
    CSS
    A * B { background-color:aquamarine;}
    Output
    • One
      • Two
        • Three
    HTML
    <ul>
       <li>One</li>
       <ul>
         <li<Two</li>
         <ul>
           <li<Three</li>
         </ul>
       </ul>
    </ul>
    Comment
    ul>li+ul>li { } produces the same effect
Notes
descendent combinator (space)
Selects nodes that are descendents of first element
div span will select all span elements that are inside a div element
Child combinator ( > )
Selects nodes that are direct children of the first element
ul>li will match all ul elements that are nested directly inside a ul element
General sibling combinator ( ~ )
Selects siblings
The second element follows the first (though not necessarily immediately)
and both elements have the same parent
p~span will match all span elements that follow a p, immediately or not
Adjacent sibling combinator ( + )
Selects adjacent siblings
The second element must directly follow the first
h2+p will match all p elements that directly follow a h2 element
[attr]
Elements with the attribute name of attr
[attr=value]
Element with the attr name of value
[attr~=value]
attr value is a whitespace-separated list of words one of which equals value
[attr|=value]
attr value can be exactly value or can begin with value
[attr^=value]
attr whose value is prefixed with value
[attr$=value]
attr is suffixed with value
[attr*=value]
attr whose value contains at least one of value within the string
[attr operator value i]
Inclusion of i or I makes comparison case-insensitive
A * B
Selects grandchildren (and great-grandchildren, great-great-grandchildren,etc)
Spaces are descendant selectors and are important
