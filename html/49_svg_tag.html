Notes
elements in xml
All elements must be closed: <rect />
Comments in xml
Same as in html: <!--comment-->
?xml
no spaces between ? and xml
<?xml version="1.0" encoding="utf-8" ?>
viewport
Like window
Set with width and height attributes in the svg tag
viewBox
viewBox is like a telescope
Can use to 'pan' and 'zoom'
Add viewBox attribute to the svg element
Syntax is four space-separated numbers
  viewBox = "x y width height"
The first two numbers control 'panning'
The second two numbers control 'zoom'
To pan-right (pan-left), increase (decrease) x
To pan-down (pan-up) increase (decrease) y
Make width and height larger than those of the viewport to zoom out
Make these last two parameters smaller than those of viewport to zoom in
object
Note that object takes the data attribute
rect
x and y are the (x, y) coordinates of the rectangle, relative to the top left corner of the svg viewport
rx and ry signify rounded corners
Use width and height to specify dimensions
circle
cx and cy are the (x,y) coordinates
r is the radius
polyon
Use points="" attribute to give a space-separated list of (x,y) coordinates
text
The y coordinate refers to bottom of the text.
Text is anchored by default at (x,y).
This may be changed for x coordinate with the text-anchor attribute, taking values of start, middle and end
To make text vertical, one possibility is: writing-mode:tb, where tb is 'top-to-bottom'
glyph
A visual representation of letter or symbol
font
A collection of glyphs
character
A binary representation of letter or symbol
