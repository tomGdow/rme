Self-Closing (void) Elements
xx
void (self-closing)
Void elements cannot have any content.
Void elements may be self-closing but the / is optional.
Thus <img> or <img /> but not <img></img>
Void elements include br, hr, input, link, base, meta, track, source,
Syntax
  1. <img>
  2. <img />
