Numeric Character Reference
In HTML it is possible to represent characters from the whole of Unicode by a numeric character reference (wikipedia)
A character reference is a sequence of characters that explicitly spell out the Unicode code point of the character being represented.
A character reference takes the form &#N;
N is either a nnnn decimal number for the Unicode code point
or a hhhh hexadecimal number, in which case it must be prefixed by x
Named character reference
Character entities can be included in an HTML document via the use of entity references
Take the form &EntityName;
Entity
Strictly speaking, a named character reference (eg &lt;)
Syntax
Numeric Character Reference Syntax
Syntax Comment Example Character
&name; Entity Name &euro; €
&#nnnn; nnnn is unicode code point as decimal &#8364; €
&#xhhhh; hhhh is unicode code point as hexadecimal &#x20AC; €
