p tag
The <p> tag defines a paragraph.
Browsers automatically add a single blank line before and after each <p> element.
Paragraphs are block-level elements
A p tag notably will automatically close if another block-level element is parsed before the closing </p>. See “Tag omission” note.

Notes
Tag Omission
The start tag is required.
The end tag may be omitted if the <p> element is immediately followed by:
<address>, <article>, <aside>, <blockquote>, <div>, <dl>, <fieldset>, <footer>, <form>, <h1>, <h2>, <h3>, >h4>, <h5>, <h6>, <header>, <hr>, <menu>, <nav>, <ol>, <pre>, <section>, <table>, <ul>
or another <p> element,
or if there is no more content in the parent element and the parent element is not an <a> element.
