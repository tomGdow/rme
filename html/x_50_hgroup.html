hgroup
The HTML hgroup element represents a multi-level heading for a section of a document.
It groups a set of <h1>–<h6> elements.
The <hgroup> element has been removed from the HTML5 (W3C) specification, but it still is in the WHATWG version of HTML.
It is partially implemented in most browsers, though, so is unlikely to go away.
The <hgroup> element allows the primary heading for a document section to be grouped with any secondary headings—such as subheadings or alternative titles—to form a multi-level heading.
Syntax
  1. code

Examples 
<hgroup>
    <h1>Calculus I</h1>
    <h2>Fundamentals</h2>
</hgroup>
