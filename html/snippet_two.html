:nth-child

matches elements based on their position in a group of siblings.

syntax

The nth-child pseudo-class is specified with a single argument that describes a pattern for matching element indices in a list of siblings.

Element indices are 1-based (but n is zero based)

odd

even

An+B Represents elements in a list whose indices match those found in a custom pattern of numbers, defined by An+B, where: A is an integer step size, B is an integer offset, n is all positive integers, starting from 0. It can be read as the An+Bth element of a list.

Examples

:nth-child(odd)or :nth-child(2n+1)

:nth-child(even) or tr:nth-child(2n)

:nth-child(7) Represents the seventh element.

:nth-child(5n) Represents elements 5, 10, 15, etc. The first one to be returned as a result of the formula is 0 [=5x0], resulting in a no-match, since the elements are indexed from 1, whereas n starts from 0.
