definition/description list
The dl element encloses dt and dd key-value pairs
Syntax
html
<dl>
<dt></dt>
<dd></dd>
</dl>
Examples
Example 1
<dl>
<dt> What is HTML5?</dt>
<dd>The latest HyperText markup language</dd>
<dt>When can I use it?</dt>
<dd>Right now!</dd>
</dl>
Example 2
<dl>
    <dt>Firefox</dt>
    <dt>Mozilla Firefox</dt>
    <dt>Fx</dt>
    <dd>
      A free, open source, cross-platform,
      graphical web browser developed by the
      Mozilla Corporation and hundreds of
      volunteers.
    </dd>
  
</dl>
Example 3
 <dl>
    <div>
        <dt> Last modified time </dt>
        <dd> 2004-12-23T23:33Z </dd>
    </div>
    <div>
        <dt> Recommended update interval </dt>
        <dd> 60s </dd>
    </div>
    <div>
        <dt> Authors </dt>
        <dt> Editors </dt>
        <dd> Robert Rothman </dd>
        <dd> Daniel Jackson </dd>
    </div>
</dl>
Notes
degeneracy
Multiple dd elements may be associated with a single dl element, or multiple dl elements may be associated with a single dd element.
mnemonic
ltd (dl-dt-dd)
closing tags
the closing tags </dt> and </dd> are optional.
What is permitted
dl→dt→dt
dl→dt→dt →dd→
No! dl→dd
