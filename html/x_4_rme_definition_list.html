definition/description list
The dl element encloses dt and dd key-value pairs
Syntax
html
<dl>
<dt></dt>
<dd></dd>
</dl>
Examples
html
<dl>
<dt> What is HTML5?</dt>
<dd>The latest HyperText markup language</dd>
<dt>When can I use it?</dt>
<dd>Right now!</dd>
</dl>
Notes
degeneracy
Multiple dd elements may be associated with a single dl element, or multiple dl elements may be associated with a single dd element.
mnemonic
ltd (dl-dt-dd)
closing tags
the closing tags </dt> and </dd> are optional.
