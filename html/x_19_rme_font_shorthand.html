font
The font CSS property is a shorthand for font-style, font-variant, font-weight, font-stretch, font-size, line-height and font-family
Alternatively, it sets an element's font to a system font.
Syntax
  1. code

Examples
  1. code
Notes
caption (font)
The CSS declaration font: caption invokes the operation system font
On Windows, this is Segoe UI
Initial value reset
As with any shorthand property, any individual value that is not specified is set to its corresponding initial value
Must Include
font must include values for font-size and font-family
Last Value
font-family must come last (as comma-separated list)
Order Rule One
font-style, font-variant and font-weight must precede font-size
Order Rule Two
line-height must immediately follow font-size preceded by "/" ("16px/3", for example)
font-variant
May only take the values of normal and small-caps
