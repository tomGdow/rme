Rule One
If @media (min-width) ≤ viewport-width (meta tag defined), then on mobile device, the media query will work.
Rule Two
If @media(max-width) ≥ viewport-width (meta tag defined), then on mobile device, the media query will work.
Rule Three
Rule One and Rule Two do not apply to browsers, and apparently do not apply to iPad (768px)
Rule Four
For @media(min-width:value), the media query will work (in browser) for values greater than value
Rule Five
For @media(max-width:value), the media query will work (in browser) for values less than value
Syntax
  1. code
