Position Absolute
An absolutely positioned element is one whose computed position is either absolute or fixed
top, right, bottom, left properties specify offsets from edges of containing block
An an element with position: absolute is removed from normal document flow
No space is created for element in page layout
Positioned relative to its closest positioned ancestor (an element that is statically positioned), if any.
If there is no positioned ancestor, the element is positioned relative to initial containing block
Syntax
css

.absolute {
    position: absolute;
    top: 3vw;
    left: 5vw;
}

js 

    function toggleClass(id,toggleClass,toggleMsg) {
        if (document.getElementById(id)) {
            document.getElementById(id).addEventListener('click', function () {
                if (this.classList.contains(toggleClass)) {
                    this.classList.remove(toggleClass);
                    this.innerHTML = "Static";

                } else {
                    this.classList.add(toggleClass);
                    this.innerHTML = toggleMsg;

                }
            });
        }
    }
Notes
auto/no attr specified
An element with position:absolute but where none of the position attributes have been assigned a value other than auto behaves as if it were assigned position:static, although it is absolutely positioned.
position:absolute; top:auto for example, behaves as a normal static element
