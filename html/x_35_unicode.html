Unicode
In Unicode, a letter maps to a code point
A code point is a theoretical concept, a 'platonic ideal' (Joel Spolsky)
How a code point is represented in memory is another, entirely separate, problem (encoding)
Every letter in every alphabet is assigned a magic number or code point by the Unicode consortium (Joel Spolsky)
U+0639
The U+ means “Unicode”
The numbers are hexadecimal
UTF-8 is one example of character encoding
An HTML document is a sequence of Unicode characters (wikipedia)
Syntax
  1. code

Examples
JavaScript
document.getElementById('myid').innerHTML="hello \u{0061}\u{0062}" // "ab"
document.getElementById('myid').innerHTML="\u{0048} \u{0065} \u{006C} \u{006C} \u{006F}" // hello
