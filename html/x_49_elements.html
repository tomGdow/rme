Elements
There are six different kinds of elements:
  1. void elements
  2. template element
  3. raw text elements
  4. escapable raw text elements
  5. foreign elements
  6. normal elements
Table Element types

Notes
raw text
Raw text elements can have text, though it has restrictions
escapable raw text
Escapable raw text elements can have text and character references, but the text must not contain an ambiguous ampersand. (There are also further restrictions)
Normal
Normal elements can have text, character references, other elements, and comments, but the text must not contain the character U+003C LESS-THAN SIGN (<) or an ambiguous ampersand.
