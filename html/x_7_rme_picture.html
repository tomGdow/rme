picture
xx
Syntax
html
<picture>
  <source media="" srcset="" sizes="" type="">
  <img srcset="" alt="">
</picture>
srcset
required
Accepts a single image file-path or comma-delimited list of image file-paths with pixel-density descriptors
media
optional
accepts any valid media query
sizes
optional
sizes="(max-width: 30em) 100vw, (max-width: 50em) 50vw"
type
optional
mime type
Examples
Opening example
<picture>
  <source srcset="./image/kitten-large.png" media="(min-width:600px)">
  <source srcset="./image/kitten-medium.png" media="(min-width: 500px">
  <img src="./image/kitten-small.png" alt="Picture of Kitten">
</picture>
example 2
<picture>
  <source
     media="(min-width: 650px)"
     srcset="images/kitten-stretching.png,
         images/kitten-stretching@1.5x.png 1.5x,
         images/kitten-stretching@2x.png 2x">
  <source
    media="(min-width: 465px)"
    srcset="images/kitten-sitting.png,
        images/kitten-sitting@1.5x.png 1.5x
        images/kitten-sitting@2x.png 2x">
  <img
    src="images/kitten-curled.png"
    srcset="images/kitten-curled@1.5x.png 1.5x,
        images/kitten-curled@2x.png 2x"
    alt="a cute kitten">
<picture>
Notes
img
The img element must be present, and should appear last ( after the final source element )
The image defined in a matching source element is substituted into the src attribute of the image element
width and height are defined by the img element values
